v0.78.0: ScriptdImporter の改修¶

主に、1.0 向けと glb/gltf の ScriptdImporter の改修です。

v0.78.0 (2021-06-23)¶

  • commit: https://github.com/vrm-c/UniVRM/releases/tag/v0.78.0

  • milestone: https://github.com/vrm-c/UniVRM/milestone/40?closed=1

Full Changelog

バグ修正¶

  • Feature10/fix sparse #1049 (ousttrue)

Exporter¶

  • Export 時に default 値の TRS を出力しない #1061 (ousttrue)

ScriptedImporter(gltf/glb vrm-1)¶

  • ImporterContext.AnimationClips に External か否かの情報を追加 #1050 (ousttrue)

  • Feature10/remap editor #1053 (ousttrue)

  • external_object_map に null が入っているときがある(対象を削除したとき) #1047 (ousttrue)

1.0¶

  • vrm0 の時だけ migration checkbox #1062 (ousttrue)

  • [1.0]MetaEditorの復旧 #1058 (ousttrue)

  • Feature10/impl firstperson list #1052 (ousttrue)

その他のChangelog¶

  • fix using #1066 (ousttrue)

  • Allow referencing additional chunks. #1065 (Santarh)

  • Add migration code about legacy roughness texture behaviour. #1064 (Santarh)

  • Public ITextureDeserializer #1063 (Santarh)
